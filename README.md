# todolist  
一个简洁的待办事项管理工具，帮助你高效跟踪和完成任务。  

## 功能特点  
- 添加新的待办事项（支持标题、截止时间）  
- 标记任务为“已完成”或“未完成”  
- 查看所有任务列表（按时间排序）  
- 删除不需要的任务  

## 如何运行  
### 环境要求  
- Java 11

### 启动步骤  
1. 克隆仓库到本地：  
   ```bash  
   git clone https://github.com/EtchoCat/todolist.git
2.进入项目内部
   cd todolist 
  
3.运行java程序

   javac TodoList.java

   java TodoList
